Formatting Help

This generator uses a BBCode-style tag format. Tag are written like so:

[name attr1=<value> attr2=<value>]contents[/name]
  • [name <attributes...>] is the opening tag.
  • attr1 and attr2 are attributes, and they allow you to provide more information to the generator.
  • [/name] is the closing tag.

Certain tags will only require one attribute, and they can be written as such: [name=<value>]contents[/name]

This expands into [name value=<value>]contents[/name].

Additionally, certain tags are leaf tags - tags that do not allow content.
These can be written without a closing tag, as such: [name <attributes...>]

Here is a list of all the tags this generator currently supports:

  • Normal new lines are ignored. To add a new line, use a br tag.
    br nodes are leaf tag - nodes without any contents, and can be written simply as [br].

  • To make text bold, wrap it in a b tag.

  • To make text italic, wrap it in a i tag.

  • To add an underline to text, wrap it in a u tag.

  • To strike through text, wrap it in a s tag.

  • To make text appear in superscript, wrap it in a sup tag.

  • To make text appear in subscript, wrap it in a sub tag.

  • To change the text color, wrap it in a color tag. You can also use a c tag, as it maps to color.

    Color nodes are written like so:
    [color=<value>]contents[/color]

    <value> can be in one of these formats:
    • #RRGGBB - hex format
    • #RGB - 3-digit hex format, expands to #RRGGBB
    • rgb(red, green, blue) - CSS format
    • palette(index), or pal(index) - palette color reference - every game definition has its own palette

  • To change the text size, wrap it in a style tag:
    [style size=<+/-number>]contents[/style]

    <+/-number> can be either +number or -number, and controls how bigger or smaller the wrapped text is from other text.

    You can also add a color attribute to change the text's color, which uses the same format as the color tag's value.

  • You can also add miscellaneous text effects by wrapping the text in a gimmick tag:
    [gimmick fill=<fill> flip=<flip>]contents[/gimmick]

    <fill> can be one of:
    • color, or colour - default, uses text color as defined by a color/style tag
    • rainbow - makes text rainbow!
    <flip> can be one of:
    • none - default
    • horizontal, or horiz/horz/h
    • vertical, or vert/v
    • both, or h&v/hv
